This directory contains the birdland data files directories:

    Book-Lists 
    Canonical
    Index-Sources
    Music-Index
    YouTube-Index

Also includes packaging information for bundles with:

    PyInstaller 
    Nuitka

The symbolic link 'src' is for packaging with PyInstaller and is omitted
from other builds.  PyInstaller failed when 'src' was 'birdland'.
